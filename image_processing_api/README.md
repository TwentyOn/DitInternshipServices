# API для обработки изображений

## Функционал

Данное API предназначаено для быстрой обработки больших пакетов 
изображений (измнение формата, качества, разрешения) "в один клик"
На вход принимает файлы форматов zip, webp, jpg, jpeg, svg, ai, eps.
и соотвествующие параметры для обработки.

При обработке zip-архива, обрабатываются только файлы-изображения
допустимых форматов. При обработке изначальная структура файлов и папок
остаётся неизменной. После обработки возвращается zip-архив с 
обработанными изображениями.

Сценарий обработки:
- валидация формата файла
- обработка файла
- загрузка обработанного файла в удалённое S3-хранилище
- возврат клиенту ответа с ссылкой на скачивание обработанного файла


## Структура проекта:

Структура проекта исполнена в соответствии с базовой структурой
для django-приложений.

Кастомные модули:
- [FileProcessor.py](FileProcessor.py) - модуль для обработки входящих 
файлов. Содержит 2 класса:
  - FileProcessor - класс для работы с входным файлом, реализует
  логику обработки в зависимости от типа файла (одиночное изображение 
  или zip-архив)
  - ImageProcessor - класс для обработки изображений, получает на вход
  изображение и параметры для обработки. Реализует обработку для 
  растровых и векторных (svg) типов графики
